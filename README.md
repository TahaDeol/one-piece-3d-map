# One Piece Interactive World Map

An interactive 3D globe web application built with CesiumJS, 
allowing users to explore the One Piece world with fly-to animations, 
location markers, and detailed island information.

## Tech Stack
- CesiumJS — 3D globe rendering
- Vanilla JavaScript — application logic
- Cloudflare R2 — tile hosting
- Vercel — deployment

## Features (In Progress)
- [ ] 3D globe with One Piece map texture
- [ ] 200+ location markers
- [ ] Fly-to camera animations
- [ ] Location search
- [ ] Straw Hat route overlay
- [ ] Arc/sea filtering
